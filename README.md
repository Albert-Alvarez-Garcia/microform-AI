# microform-AI
_MICROFORM // Dynamic JSON Renderer

English | Español
English
Microform is a minimalist, brutalist-inspired tool designed to generate dynamic web forms from JSON schemas. It integrates Google Gemini's AI to convert natural language prompts into structured, functional UI components in seconds.
<img width="3416" height="1191" alt="image" src="https://github.com/user-attachments/assets/6045c511-a964-424b-9071-0fed8dcb9dd6" />

Key Features
AI-Powered Generation: Converts natural language prompts into valid JSON forms using Google Gemini.
Reactive Rendering: Real-time form updates as you edit the JSON schema.
Dynamic Inputs: Supports text, number, select, radio, and textarea.
Makefile Automation: Simple environment setup and server management.
Installation & Quick Start
Ensure you have python3 and make installed.
Clone the repository:
git clone https://github.com/Albert-Alvarez-Garcia/microform
cd microform


Install dependencies:
make install


Run the server:
make run

The server will be live at http://localhost:8000.
